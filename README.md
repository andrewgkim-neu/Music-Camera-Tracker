# Music-Camera-Tracker
[YouTube Demo](https://youtu.be/j9A5S8RB8Es?si=BqLNssocx58mbWDR) \
## Features

- **Real-time hand tracking** using MediaPipe
- **Play/Pause control** with hand gestures
- **Resume playback** from last paused moment
- **Automatic looping** of any song loaded
- **Visual feedback** showing hand landmarks and gesture detection
- **Supports multiple audio formats** (MP3, WAV, OGG)

## How It Works

1. Make a circle with your hand by touching your thumb to your index finger
2. Music starts playing immediately
3. Hold the gesture to keep the music playing
4. Release the gesture to pause the music
5. Make the gesture again to resume from where you paused
